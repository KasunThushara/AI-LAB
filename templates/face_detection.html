Face Landmark Detection

About Face Landmark Detection

This system uses MediaPipe with Hailo NPU acceleration to detect facial landmarks in real-time. MediaPipe is an open-source framework from Google for building multimodal machine learning pipelines.

Key Features:

  • Detects 468 facial landmarks with high precision
  • Runs efficiently on Hailo-8 NPU hardware
  • Real-time performance even on edge devices
  • Enables applications like AR filters, facial analysis, and more
