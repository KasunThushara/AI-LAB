Hand Landmark Detection

About Hand Landmark Detection

This system leverages MediaPipe with Hailo NPU acceleration to detect hand landmarks in real-time. MediaPipe provides efficient on-device machine learning solutions for complex tasks.

Key Features:

  • Detects 21 hand landmarks per hand
  • Optimized for Hailo-8 NPU hardware acceleration
  • Low-latency performance suitable for interactive applications
  • Enables gesture recognition, sign language translation, and more
